# udit-ui Documentation

This is the documentation website for udit-ui components, built with Next.js.

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the documentation.

## Build

```bash
pnpm build
pnpm start
```

## Deployment

This project is configured for Vercel deployment. The build configuration is in `vercel.json`.
